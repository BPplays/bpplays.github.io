My Elden Ring Steam Controller Config

Published: 2026年06月10日

This config expects A to be roll and B to be jump, and for mouse lock-on target change sensitivity to be lowest


Download the config here and put it in <steam folder>\steamapps\common\Steam Controller Configs\<user id>\config\1245620.

About the config

The A button uses Start Press to be act on press instead of on release for lower latency (see Carmack's tweet) this makes A roll only, and L4 for running


this config is for A to be roll only (having A roll on press instead of on release is lower latency) and L4 to be for running, B is only for menus and R4 is for jumping, left touchpad is for pouch.

the right joystick acts as mouse to work around disabling auto camera rotation being broken, in the map screen hold B then the bumper become zoom in the map,
holding B also turns the left touchpad into a shortcut for gestures (only works when outside a menu)
